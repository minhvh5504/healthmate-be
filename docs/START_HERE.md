# 🚀 START HERE - OMMS Backend cho Team Frontend

## 👋 Chào mừng Team Frontend!

Đây là package backend OMMS đã được cấu hình sẵn để bạn chạy trên máy local và tích hợp với Frontend.

---

## 📖 BẮT ĐẦU TỪ ĐÂY

### 🎯 Bước 1: Đọc hướng dẫn
👉 **[QUICK_STARTUP.md](./QUICK_STARTUP.md)** 👈

File này có **TẤT CẢ** hướng dẫn bạn cần, bằng Tiếng Việt, rất dễ hiểu!

### 🚀 Bước 2: Chạy backend
Sau khi đọc xong, chạy script:
- **Windows**: Double-click `quick-startup.bat`
- **Mac/Linux**: Mở Terminal, gõ `./quick-startup.sh`

### ✅ Bước 3: Bắt đầu code
Backend sẽ chạy tại: `http://localhost:8080`
